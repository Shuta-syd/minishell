## Rules for commit messages
### Labels
- [add] : add files, functions
- [mod] : modify something
- [fix] : fix bugs
- [ref] : refactoring
- [norm] : adjust norminette
- [rm] : remove files, functions
- [...] : commitments not worth mentioning
- [init] : initialize repository
